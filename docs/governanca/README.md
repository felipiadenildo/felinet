# 00 — Governança

Documentos de gestão e governança do TCC. Não são conteúdo da monografia em si, mas o **rastro de decisões**, **diretrizes vinculantes** e **acordos** que orientam todo o resto.

| Arquivo | Função |
|---|---|
| `CHANGELOG_revisao_v2.md` | Histórico consolidado da revisão v2 (Fases 1.1 a 1.6) |
| `diretrizes_projeto.md` | Diretrizes vinculantes acordadas com o Pesquisador |
| `alinhamento_projeto_base.md` | Alinhamento entre o PDF original do projeto e as revisões |
| `glossario_notas_rodape.md` | Glossário de termos (vira notas de rodapé na monografia) |
| `fontes_e_lacunas.md` | Lacunas de fontes pendentes e roteiros de busca manual |
| `notas_pontos_futuros.md` | Ideias e pontos abertos para iterações futuras |

**Política de mudanças**: toda alteração relevante neste diretório deve gerar entrada no CHANGELOG.
