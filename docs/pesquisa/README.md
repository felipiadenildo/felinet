# 01 — Pesquisa

**Fonte textual** do TCC. Cada Markdown aqui é a base de um capítulo (ou seção) da monografia em LaTeX. Quando a Fase 5 começar, o conteúdo destes MDs migra (com adaptações) para `02_latex/capitulos/`.

## Estrutura

- **`etapa1_fundamentacao/`** — Capítulos de fundamentação teórica (A1.1 a A1.6).
- **`etapa2_engenharia/`** — Capítulos de concepção arquitetural e pipeline (A2.0, B1 a B6).
- **`_buscas_brutas/`** — JSONs de buscas executadas para rastreabilidade (não vai para a monografia).

## Convenções

- Cada MD começa com cabeçalho `# Título — descrição`.
- Citações inline em formato Markdown: `[Nome (ano)](url)`.
- Listas de fontes ao final de cada parte.
- Placeholders em `[PLACEHOLDER-*]` ou `[~estimativa]`.
