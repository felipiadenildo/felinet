# 09 — Artefatos de entrega

Saídas finais consolidadas do TCC. Esta pasta é o **deliverable**.

Esperado ao final:

- `tcc_final.pdf` — monografia compilada (cópia de `02_latex/main.pdf`)
- `tcc_final_bibtex.bib` — base bibliográfica final
- `slides_defesa.pdf` — apresentação da defesa
- `poster_resumo.pdf` (opcional) — pôster científico
- `manual_voluntario_aex.pdf` — anexo destacável para campo
- `repositorio_github_release.md` — link do release com tag de versão

## Política

- Apenas PDFs/arquivos finais entram aqui. Working files ficam nas pastas de origem.
- Cada entrega tem tag Git correspondente (ex.: `v1.0-defesa`, `v1.1-pos-banca`).
