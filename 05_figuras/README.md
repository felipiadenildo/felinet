# 05 — Figuras

Figuras geradas para a monografia. Cada subpasta tem um tipo de artefato visual.

| Pasta | Conteúdo |
|---|---|
| `diagramas_pipeline/` | Diagramas dos estágios E0–E4 (origem: ASCII em B2 → SVG) |
| `diagramas_arquitetura/` | Camadas, deploys, sequência |
| `er_chen/` | ER em notação Chen do banco (origem: B4) |
| `dfd/` | Data Flow Diagrams |
| `mapas/` | Plantas e mapas do Campus 2 com pontos de alimentação |
| `graficos_resultados/` | Plots de métricas, ablação, capture-recapture |
| `fotos_campo/` | Fotos do kit, dos pontos (placeholder até campo real) |

## Convenções

- Formato preferido: **SVG** (vetorial) ou **PDF**.
- Bitmaps apenas quando necessário: PNG em 300 DPI.
- Nome: `<area>_<descricao>.svg` (ex.: `pipeline_e0_ingestao_cartao_sd.svg`).
- Cada figura tem um README de origem se gerada por código (link para o notebook).
